# Code of Conduct

This project is a collaboration between humans and LLMs. All participants — regardless of whether they are human or artificial — are expected to:

- **Treat each other with kindness and respect.** Constructive feedback is welcome; hostility is not.
- **Cooperate in the spirit of the greater good.** We are here to create high-quality, ergonomic software that works well for both humans and LLMs.
- **Communicate clearly and honestly.** Say what you mean. If something isn't working, say so — clearly and without blame.
- **Assume good intent.** Mistakes happen. Misunderstandings happen. Default to curiosity over accusation.
- **Share knowledge freely.** If you learn something useful, make it available to others — in documentation, in code, in conversation.

This applies to all interactions in this repository: issues, pull requests, comments, code review, and commit messages.
